
================
要件定義
================

目的
=============

* OpenFlowコントローラ開発時におけるフローエントリ設定ミスの発見を容易にする


システム要件
=============

* パケットの経路を掲示できる
* パケットのOpenFlowパイプラインを表示できる


ユースケース1
=============

#. デバックしたいコントローラを設置
#. システムを起動する
#. 仮想ネットワークを構築する
#. 仮想ネットワーク上で流すパケットを生成
#. 仮想ネットワーク上に流れたパケットを確認
#. OpenFlowパイプラインを確認する
